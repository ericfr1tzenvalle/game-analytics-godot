# Game Analytics

### Objetivo

Desenvolver um jogo de cartas em Godot capaz de registrar os eventos e decisões realizadas durante a partida.
Os dados coletados vão ser utilizados para analisar o comportamento do jogador, `identificar padrões`, encontrar possíveis problemas de balanceamento e avaliar o impacto das cartas, inimigos e mecânicas de jogo.

`Game Analytics` tem como principal objetivo unir duas áreas que eu gosto que é dados e desenvolvimento de jogos.

### Ciclo principal

-> Desenvolvimento do jogo -> Coleta de dados 
-> Tratamento e Análise -> Documentação dos resultados 
-> Balanceamento -> Novos Testes
